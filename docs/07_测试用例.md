# 本地学校学生管理网页仿真系统：测试用例（精简版）

---

## 1. 测试环境

**本地环境**：
- 操作系统：Windows 10/11
- Node.js：18.x 或 20.x
- 浏览器：Chrome / Edge
- 后端端口：3000
- 前端端口：5173

**测试账号**：
| 角色 | 账号 | 密码 |
|---|---|---|
| 管理员 | admin | 123456 |
| 教师 | teacher001 | 123456 |
| 教师 | teacher002 | 123456 |
| 学生 | student001 | 123456 |
| 学生 | student002 | 123456 |

**测试数据最低要求**：
- 管理员 1、教师 2、学生 30+
- 专业 3+、班级 4+、课程 10+
- 成绩/考勤 30+、公告 5+、奖惩 5+
